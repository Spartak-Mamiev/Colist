# App name

CoList – Shared Grocery App

# Description

CoList is a real-time shared grocery list application built with React and Supabase.
It allows couples and roommates to collaboratively manage household shopping with instant updates and secure list sharing.

# The problem it is solving

Managing shared grocery lists in households is often fragmented across notes apps, messaging apps, or paper lists. CoList provides a real-time, structured, and collaborative solution.

# Stack

FRONTEND:

- Vanilla Javascript
